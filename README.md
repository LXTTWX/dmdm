# 随机点名系统 - 特殊规则版本

## 项目简介



## 主要功能

### 🎯 核心点名功能
- **多种点名算法**：公平随机、权重随机、最少点名优先
- **滚动动画**：炫酷的学生姓名滚动效果
- **实时统计**：点名次数、频率分析
- **历史记录**：完整的点名历史追踪

### 👨‍🎓 特殊规则管理
- **学生状态管理**：正常状态/特殊排除状态
- **智能过滤**：非公平模式下自动排除特殊学生
- **公平保护**：公平随机模式下保持原有概率
- **操作日志**：记录所有配置变更和时间戳

### 📊 数据管理
- **班级管理**：支持多班级管理
- **学生管理**：批量导入、导出功能
- **数据备份**：完整的数据备份和恢复
- **统计分析**：详细的点名统计图表

### 点名模式说明
- **公平随机**：所有学生（包括特殊状态学生）均有相同概率被选中
- **权重随机**：基于历史点名次数的加权随机，特殊状态学生被排除
- **最少点名优先**：优先选择被点名次数最少的学生，特殊状态学生被排除
